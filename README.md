# Work Hays

> Work Hays https://workhays.netlify.app (Currently in development)

## About

Work Hays is a job postings website for the Hays, KS area and surrounding regions. This is a free website for employers to post their open positions & for potential employees to discover.

## Getting Started

Follow this guide to get started developing & contributing. Pull requests welcome.

1.  🍴 Fork this repository

2.  💾 Clone your fork to your local environment `git clone https://github.com/{{ your github username }}/workhays-next`

3.  📦 Install dependencies `yarn install`

4.  🌎 Start local dev server `yarn dev`

5.  🌎 Start up local development server `npm run dev` on `http://localhost:3000`

## Technical Overview

This is a Typescript/React application using [Next.js](https://nextjs.org/) framework and [Tailwind CSS](https://tailwindcss.com/) for styling. [Supabase](https://supabase.io/) Supabase is used to store jobs and details, and authentication for employers to sign in and manage their openings. The site is currently hosted on [Netlify](https://www.netlify.com/), and using Netlify for serverless functions and form capturing.
